Sugar Oppa

Sugar Oppa is a full-stack app aimed at making budgeting and creating financial goals more simple. The user can log in securely and gain access to tools such as a chart to show your spending, Technologies my team and I used included HTML, CSS, Javascript, Node.js, PostgresSQL, Express.js, and a REST API.

React Blog

This is a full-stack react blog that incorporates react hooks and functional components. It allows uses to add new posts and comments to the posts.

Memory Game

This memory game is a online game where the user can practice their memorization skills by flipping cards to match sets of two. It was built using React and allowed me the opportunity to discover React for the first time, differentiate JSX and HTML expressions, and practice using style components.
